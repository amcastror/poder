# Prompt Base General

```text
Crear una imagen abstracta minimalista para una presentacion web tipo keynote sobre SaaS, IA, workflows, automatizacion y estrategia empresarial.

Estilo visual: abstracto, sobrio, premium, minimalista, oscuro, de alto contraste, con composicion limpia y pocos elementos. Debe sentirse como una visualizacion estrategica de sistemas, flujos operativos, riesgo, confianza y toma de decisiones, no como una ilustracion decorativa ni como una interfaz de producto.

Paleta: fondo negro o gris carbon profundo, lineas y volumenes en grises oscuros, blanco tenue para puntos de enfasis, y acento naranja calido similar a rgb(236,129,18). Mantener la paleta actual del sitio. Evitar colores brillantes adicionales.

Composicion: formato horizontal 16:10, pensada para acompanar texto en un layout de presentacion. La imagen debe funcionar como apoyo visual, no competir con el contenido. Dejar respiracion, margen visual y una zona de composicion clara. Usar formas geometricas, nodos, capas, rutas, conexiones, estructuras, nucleos, barreras, flujos o arquitecturas abstractas.

Tema: representar de forma abstracta la transicion desde software basado en pantallas hacia sistemas que entienden contexto, deciden y ejecutan workflows. Puede combinar metaforas de red, orquestacion, automatizacion, presion competitiva, proteccion estrategica, datos propietarios y confianza operativa.

Restricciones: sin personas, sin rostros, sin manos, sin oficinas, sin laptops visibles, sin pantallas de producto, sin dashboards, sin UI ficticia, sin mockups de software, sin logos, sin marcas, sin texto, sin letras salvo cuando se pida explicitamente para el framework PODER. No usar estetica de SaaS generico, tarjetas flotantes, paneles de interfaz, graficos de marketing, ilustracion cartoon, 3D colorido, fondos con blobs decorativos ni gradientes llamativos.

Calidad: imagen nitida, elegante, editorial, con sensacion de keynote ejecutiva. Iluminacion controlada, profundidad sutil, lineas precisas, geometria refinada, sin ruido visual.
```
